CSS3.0特性

1. 背景图片大小设定(background-size)

​ 通过使用background-size 来规定背景图片的尺寸

​ background-size要写在background-image后面，否则background-size不会生效

​ background-size 的值有：

  • background-size：contain; //背景成比例，按照图片最大边进行适配，图片完全显示在盒子里，少的部分出现白边
  • background-size ：cover; // 背景图成比例，按照图片最小边进行适配，多余的裁切
  • background-size ：100px 100px; // 调整图片到指定大小；
  • background-size ：100%; 会按容器比例撑满，导致图片失真。

2. 背景透明颜色(rgba)

​ 通过使用rgba(xx,xx,xx,xx,xx)来设置，其中前三项与rgb(xx,xx,xx)一样为红绿蓝三种原色，后面的一项规定了透明度(alpha)，alpha参数是介于0.0(完全透明)~1.0(完全不透明之间)

3. 图像的透明(opacity)

​ 通过使用opacity参数来设定图像的透明度，opacity参数是介于0.0(完全透明)~1.0(完全不透明之间)

​ opacity在使用上应用于使用该属性的盒子内部的前景色、后景色、图片等，往往造成误伤，很多时候我们需要背景色透明度，而使用opacity则会导致文字颜色跟随改变

​ 注意：opacity会继承父元素的opacity属性,而RGBA设置的元素的后代不会继承该属性及属性值。

4. 颜色的渐变

​ 通过使用background-image:linear-gradient(Cdeg,#A 20%,#B 90%)来设置颜色渐变，由A颜色渐变到B颜色，C表示角度，未设置角度时，则默认未180deg(从上到下)，设置角度，则0deg为竖直向上，然后顺时针旋转.后面的百分号为前20%都为A颜色，而后渐变，到90%到最后全为B色，如果设置满了不会有渐变效果，即全为纯色，注意：可以设置多个颜色参与渐变

5. 个性边框

(1). 圆角(Border-radius)

​ 通过使用Border-radius属性来设置边框的圆角度，可以使用px、%、em来表示数值

  • 仅设置一个值

    ​ 设置一个数值，即对元素的四个边角设置统一的圆角弧度

    ​ 一般用作设置按钮或者获得圆形(Border-radius为正方形一半px值时即可获得)

  • 设置四个方向的值

    ​ border-radius属性其实是border-top-left-radius、border-top-right-radius、border-bottom-right-radius、border-bottom-left-radius四个属性的简写模式

    ​ 不同于padding和margin的“上、右、下、左”的顺序，border-radius采用的是左上角、右上角、右下角、左下角的顺序

    ​ 通过设置四个数值可以更加精确获得需要的形状

(2). 阴影

​ 在CSS3中，使用box-shadow来给方框添加阴影

语法

参数解释：

  • offset-x：必需，取值正负都可。offset-x水平阴影的位置。
  • offset-y：必需，取值正负都可。offset-y垂直阴影的位置。
  • blur:可选，只能取正值。blur-radius阴影模糊半径，0即无模糊效果，值越大阴影边缘越模糊。
  • spread：可选，取值正负都可。spread代表阴影的周长向四周扩展的尺寸，正值，阴影扩大，负值阴影缩小。
  • color:可选。阴影的颜色。如果不设置，浏览器会取默认颜色，通常是黑色，但各浏览器默认颜色有差异，建议不要省略。可以是rgb(250,0,0)，也可以是有透明值的rgba(250,0,0,0.5)。
  • inset:可选。关键字，将外部投影(默认outset)改为内部投影。inset 阴影在背景之上，内容之下
