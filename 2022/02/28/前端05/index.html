DIV+CSS布局

1. 块元素和行内元素的区别

​ 块元素和行内元素本质上是html规范中的概念

  • 块(block)元素的特点

    • 总是在新行上开始
    • 高度和行高以及外边距和内边距都可以控制
    • 宽度缺省是它的容器的100%，除非设定一个宽度
    • 它可以容纳行内元素和其他块元素

  • 行内(inline)元素的特点

    • 和其他元素都在一行上
    • 高，行高以及外边距和内边距不可改变
    • 宽度就是它的文字或图片的宽度，不可改变
    • 行内元素只能容纳文本或者其他行内元素

​ 注意：一般情况下，做布局使用块元素，在一行的内容使用行内元素

2. 常见的块元素

  • div
  • dl
  • form
  • h1~h6
  • hr

4. CSS盒子模型

​ 盒子模型是CSS的基石之一，在HTML页面上的每一个元素都被浏览器看成是一个矩形的盒子，这个盒子由元素的内容、填充、边框、边界组成

​ 网页就是由许多个盒子通过不同的排列方式堆积而成，早期是使用表格做布局，现在使用的是DIV+CSS布局

5. 盒子模型详细介绍

​ 标准盒子模型示意图

​ 在盒子模型中，可以单独设定任何一个元素，也可以组合起来按照上、右、下、左的顺序进行设定

​ 盒子模型的背景颜色范围为border里面

​ 元素盒子大小的计算

​ 一个元素的实际宽度=左边界 + 左边框 + 左填充 + 内容宽度 + 右填充 + 右边框 + 右边界

​ 一个元素的实际高度=上边界 + 上边框 + 上填充 + 内容高度 + 下填充 + 下边框 + 下边界

6. CSS盒子嵌套

​ 两个盒子相互嵌套是用大盒子的content来包裹小盒子

​ 在一个盒子中嵌套一个盒子，那么两个盒子边框之间的距离等于外面盒子的填充值 + 里面盒子的边界值

​ 盒子模型的特性

  • 边界值margin可为负，填充padding不可为负
  • 边框border默认值为0，即不显示
  • 盒子模型只能设置两类颜色：边框颜色和背景颜色

​ 盒子模型属性值的简写形式

  • 给出一个属性值：表示上右下左设置的全部值都为它
  • 给出两个属性值：前者表示上下边框的属性，后者表示左右边框的属性
  • 给出三个属性值：前者表示上边框的属性，中间的数值表示左右边框的属性，后者表示下边框的属性
  • 给出四个属性值：依次表示上右下左的属性

行高

​ 行内元素在浏览器中只能得到一行高度的空间，行高由line-height 属性(像素或百分比为单位)决定，如果没有设置该属性，则为内容的默认高度

display属性

​ 通过display属性可控制元素是以行内元素显示还是以块级元素显示，或者不显示，即对元素进行转换

  • display:block 当前元素以块级元素显示
  • display:inline 当前元素以行内元素显示
  • display:none 当前元素隐藏，浏览器会完全忽略这个元素(不是删除是不显示)，一般使用在下拉菜单中
  • display:inline-block 当前元素既可以控制宽高，又可以横向排列

9.浮动与浮动清除

​ 浮动主要是排版使用，可以使得原本2D的网页设计变成3D

​ 浮动可以理解为让某个div元素脱离标准流，漂浮在标准流之上，和标准流不是一个层次

浮动的设置(对于要浮动的盒子设置)

​ CSS中使用float 属性来控制浮动，其默认值为none ，如果将float 属性设置为left 或者right ，元素就会向左侧或者右侧靠紧，同时盒子宽度不再伸展，没设置宽度时会根据盒子里面的内容来确定宽度

​ 如果某块设置为浮动，那么其后面的块会自动补该块未浮动时的位置，但后面块的文字会被挤占

浮动的清除(对于浮动盒子的下面盒子设置)

​ CSS使用clear 属性来清除浮动(消除浮动对下面盒子的影响)，它的取值有left、right、both和none(默认值)，设置为left或者right表示该盒子的左边或右边不允许有浮动的对象，设置为both则表示两边都不允许有浮动对象，因此该盒子将会在浏览器中另起一行显示

10. position属性的使用

​ 一般使用position属性来对盒子模型进行定位，一般不会使用margin来设置位置，因为margin只是指外边距

​ position属性修改元素位置后，对其他元素无影响

​ position属性常用的4个值

  • static

    ​ static是position的默认属性值，忽略top、bottom、left、right

  • relative

    ​ relative表示相对定位，当 position 为 relative 的时候，如果设置了 left 、right、top、bottom 属性，它们会生效，发生偏移时的参照为 position 属性取 static 时盒子的位置

  • absolute

    ​ absolute表示绝对定位，当 position 为 absolute 的时候，元素会被移出正常文档流，并不为元素预留空间，元素相对于最近的非 static 定位祖先元素发生偏移，当元素没有非 static 定位祖先时，会相对 html 发生偏移

  • fixed

    ​ fixed表示固定定位，当 position 为 fixed 的时候，元素会被移出正常文档流，并不为元素预留空间，元素相对于屏幕视口（viewport）的 位置来发生偏移, 元素的位置在屏幕滚动时不会改变
