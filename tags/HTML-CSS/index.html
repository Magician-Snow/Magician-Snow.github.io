    CSS3.0特性

    1. 背景图片大小设定(background-size)

    ​ 通过使用background-size 来规定背景图片的尺寸

    ​ background-size要写在background-image后面，否则background-size不会生效

    ​ background-size 的值有：

    • background-size：contain; //背景成比例，按照图片最大边进行适配，图片完全显示在盒子里，少的部分出现白边
    • background-size ：cover; // 背景图成比例，按照图片最小边进行适配，多余的裁切
    • background-size ：100px 100px; // 调整图片到指定大小；
    • background-size ：100%; 会按容器比例撑满，导致图片失真。

    2. 背景透明颜色(rgba)

    ​ 通过使用rgba(xx,xx,xx,xx,xx)来设置，其中前三项与rgb(xx,xx,xx)一样为红绿蓝三种原色，后面的一项规定了透明度(alpha)，alpha参数是介于0.0(完全透明)~1.0(完全不透明之间)

    3. 图像的透明(opacity)

    ​ 通过使用opacity参数来设定图像的透明度，opacity参数是介于0.0(完全透明)~1.0(完全不透明之间)

    ​ opacity在使用上应用于使用该属性的盒子内部的前景色、后景色、图片等，往往造成误伤，很多时候我们需要背景色透明度，而使用opacity则会导致文字颜色跟随改变

    ​ 注意：opacity会继承父元素的opacity属性,而RGBA设置的元素的后代不会继承该属性及属性值。

    4. 颜色的渐变

    ​ 通过使用background-image:linear-gradient(Cdeg,#A 20%,#B 90%)来设置颜色渐变，由A颜色渐变到B颜色，C表示角度，未设置角度时，则默认未180deg(从上到下)，设置角度，则0deg为竖直向上，然后顺时针旋转.后面的百分号为前20%都为A颜色，而后渐变，到90%到最后全为B色，如果设置满了不会有渐变效果，即全为纯色，注意：可以设置多个颜色参与渐变

    5. 个性边框

    (1). 圆角(Border-radius)

    ​ 通过使用Border-radius属性来设置边框的圆角度，可以使用px、%、em来表示数值

    • 仅设置一个值

      ​ 设置一个数值，即对元素的四个边角设置统一的圆角弧度

      ​ 一般用作设置按钮或者获得圆形(Border-radius为正方形一半px值时即可获得)

    • 设置四个方向的值

      ​ border-radius属性其实是border-top-left-radius、border-top-right-radius、border-bottom-right-radius、border-bottom-left-radius四个属性的简写模式

      ​ 不同于padding和margin的“上、右、下、左”的顺序，border-radius采用的是左上角、右上角、右下角、左下角的顺序

      ​ 通过设置四个数值可以更加精确获得需要的形状

    (2). 阴影

    ​ 在CSS3中，使用box-shadow来给方框添加阴影

    DIV+CSS布局

    1. 块元素和行内元素的区别

    ​ 块元素和行内元素本质上是html规范中的概念

    • 块(block)元素的特点

      • 总是在新行上开始
      • 高度和行高以及外边距和内边距都可以控制
      • 宽度缺省是它的容器的100%，除非设定一个宽度
      • 它可以容纳行内元素和其他块元素

    • 行内(inline)元素的特点

      • 和其他元素都在一行上
      • 高，行高以及外边距和内边距不可改变
      • 宽度就是它的文字或图片的宽度，不可改变
      • 行内元素只能容纳文本或者其他行内元素

    ​ 注意：一般情况下，做布局使用块元素，在一行的内容使用行内元素

    2. 常见的块元素

    • div
    • dl
    • form
    • h1~h6
    • hr

    4. CSS盒子模型

    ​ 盒子模型是CSS的基石之一，在HTML页面上的每一个元素都被浏览器看成是一个矩形的盒子，这个盒子由元素的内容、填充、边框、边界组成

    ​ 网页就是由许多个盒子通过不同的排列方式堆积而成，早期是使用表格做布局，现在使用的是DIV+CSS布局

    ​ 盒子模型示意图

    ​ 注意：默认情况下，盒子边框是无，背景色是透明，所以在默认情况下看不到盒子，

    5. 盒子模型详细介绍

    ​ 标准盒子模型示意图

    ​ 在盒子模型中，可以单独设定任何一个元素，也可以组合起来按照上、右、下、左的顺序进行设定

    ​ 盒子模型的背景颜色范围为border里面

    ​ 元素盒子大小的计算

    ​ 一个元素的实际宽度=左边界 + 左边框 + 左填充 + 内容宽度 + 右填充 + 右边框 + 右边界

    ​ 一个元素的实际高度=上边界 + 上边框 + 上填充 + 内容高度 + 下填充 + 下边框 + 下边界

    6. CSS盒子嵌套

    ​ 两个盒子相互嵌套是用大盒子的content来包裹小盒子

    ​ 在一个盒子中嵌套一个盒子，那么两个盒子边框之间的距离等于外面盒子的填充值 + 里面盒子的边界值

    ​ 盒子模型的特性

    • 边界值margin可为负，填充padding不可为负
    • 边框border默认值为0，即不显示
    • 盒子模型只能设置两类颜色：边框颜色和背景颜色

    ​ 盒子模型属性值的简写形式

    • 给出一个属性值：表示上右下左设置的全部值都为它
    • 给出两个属性值：前者表示上下边框的属性，后者表示左右边框的属性
    • 给出三个属性值：前者表示上边框的属性，中间的数值表示左右边框的属性，后者表示下边框的属性
    • 给出四个属性值：依次表示上右下左的属性

    行高

    display属性

    ​ 通过display属性可控制元素是以行内元素显示还是以块级元素显示，或者不显示，即对元素进行转换

    • display:block 当前元素以块级元素显示
    • display:inline 当前元素以行内元素显示

    • display:inline-block 当前元素既可以控制宽高，又可以横向排列

    9.浮动与浮动清除

    ​ 浮动主要是排版使用，可以使得原本2D的网页设计变成3D

    ​ 浮动可以理解为让某个div元素脱离标准流，漂浮在标准流之上，和标准流不是一个层次

    浮动的设置(对于要浮动的盒子设置)

    ​ CSS中使用float 属性来控制浮动，其默认值为none ，如果将float 属性设置为left 或者right ，元素就会向左侧或者右侧靠紧，同时盒子宽度不再伸展，没设置宽度时会根据盒子里面的内容来确定宽度

    ​ 如果某块设置为浮动，那么其后面的块会自动补该块未浮动时的位置，但后面块的文字会被挤占

    浮动的清除(对于浮动盒子的下面盒子设置)

    ​ CSS使用clear 属性来清除浮动(消除浮动对下面盒子的影响)，它的取值有left、right、both和none(默认值)，设置为left或者right表示该盒子的左边或右边不允许有浮动的对象，设置为both则表示两边都不允许有浮动对象，因此该盒子将会在浏览器中另起一行显示

    10. position属性的使用

    ​ 一般使用position属性来对盒子模型进行定位，一般不会使用margin来设置位置，因为margin只是指外边距

    ​ position属性修改元素位置后，对其他元素无影响

    ​ position属性常用的4个值

    • static

      ​ static是position的默认属性值，忽略top、bottom、left、right

    • relative

      ​ relative表示相对定位，当 position 为 relative 的时候，如果设置了 left 、right、top、bottom 属性，它们会生效，发生偏移时的参照为 position 属性取 static 时盒子的位置

    • absolute

      ​ absolute表示绝对定位，当 position 为 absolute 的时候，元素会被移出正常文档流，并不为元素预留空间，元素相对于最近的非 static 定位祖先元素发生偏移，当元素没有非 static 定位祖先时，会相对 html 发生偏移

    • fixed

      ​ fixed表示固定定位，当 position 为 fixed 的时候，元素会被移出正常文档流，并不为元素预留空间，元素相对于屏幕视口（viewport）的 位置来发生偏移, 元素的位置在屏幕滚动时不会改变

    CSS基础

    1. CSS简介

    ​ CSS(层叠样式表)是由W3C的CSS工作组创建和维护的。它是一种不需要编译。可直接由浏览器执行的标记性语言，用于控制web页面的外观

    ​ 通过使用CSS样式控制页面各元素的属性显示，可将页面的内容与表现形式进行分离

    ​ 层叠样式表本质上也是文本，用{} 将样式内容括住，最后将所有的封装在一个文件里。页面引用这个文件即可

    ​ HTML5和CSS3是同时制作发布的

    2. CSS的优势

    • 避免使用多余的HTML代码，减少代码量

      ​ 文本本身是通过字符来计算文件量，而**在之前对于页面的修饰会通过HTML标签自带的属性进行修饰(即HTML代码的处理方式，不属于CSS)**，此时代码量大，读的时候就会慢

    • 缩短更新和维护的时间

      ​ 不使用CSS可能有些样式要反复定义，需要更换时，可能非常复杂。而且采用样式表可以使得样式与页面相分离，在修改时，更加直观、防止出错

    • 做出HTML实现不了的特效

      ​ CSS2很多功能HTML里面其实也有，但在CSS3以及之后HTML里的样式就不再更新了

    3. 认识<div> 和<span> 标签(双)

    ​ <div> 和<span> 标签是属于HTML的标签，一般这两个标签都是配合样式表使用的，这两个标签可以理解为容器标签，早期时一般用Table来当容器

    • <div> 标签(双)：

      ​ 用来为HTML文档内大块的内容提供结构和背景的元素，它为块元素，<div> 的起始标签和结束标签之间的所有内容都在该块中，一般在布局时使用

    • <span> 标签(双)：

      ​ 用来组成HTML文档的行元素，一般放内容时使用，div 也可以作为放内容

    (2). 内部样式表(在head部分加入)

    ​ 把所有的东西封装在一个样式表文件里，从外部引用这个样式表文件，当网页加载时会把所有的外部样式先加载到网页中再进行显示，网速再慢也不影响，推荐使用这个形式

    8. 伪类(里面的状态伪类)

    ​ 同一个标签，根据其不同的状态，有不同的样式。这就叫做“伪类”，伪类用冒号来表示。

    ​ 伪类选择器分为两种。

    ​ 1. 静态伪类：只能用于**超链接<a> **的样式。如下：

    • :link 超链接点击之前

    • :visited 链接被访问过之后

      注意：以上两种样式，只能用于超链接，且要写的话应该一起写上。

    ​ 2. 动态伪类：针对所有标签都适用的样式。如下：

    • :hover “悬停”：鼠标放到标签上的时候
    • :active “激活”： 鼠标点击标签，但是不松手时。
    • :focus 是某个标签获得焦点时的样式（比如某个输入框获得焦点）

    ​ 超链接的四种状态伪类

    ​ a标签有4种伪类（即对应四种状态），要求背诵。如下：

    • :link “链接”：超链接点击之前

    • :visited “访问过的”：链接被访问过之后

    • :hover “悬停”：鼠标放到标签上的时候

    • :active “激活”： 鼠标点击标签，但是不松手时。

    ​ 记住，在css中，这四种状态必须按照固定的顺序写：

    a:link 、a:visited 、a:hover 、a:active

    ​ 如果不按照顺序，那么将失效

    9. CSS颜色写法

    ​ 在CSS中，颜色的表达有一下三种方式

    1. 十六进制：如“#ff000”
    2. RGB：如“rgb(255,0,0)”
    3. 颜色名称：如“red”

    ​ 注意：一般使用十六进制表示颜色这种方式居多

    10. 背景图像

    ​ CSS关于背景图像的属性

    • background-image:url("图像路径") 设置背景图像
    • background-repeat:no-repeat 设置背景图像为不重复，默认值为重复
    • background-attachment:fixed 将图片滚动锁定为不跟随滚动
    • background-position:center 将图像位置设置为居中

    11. 文本修饰

    ​ CSS关于文本修饰的属性

    • color:blue 设置文本颜色为蓝色
    • text-align:center 设置文本位置为居中
    • text-decoration:line-through 设置文本效果为加删除线，除了删除线外还有上划线、下划线和无文字效果(none，这个可以和超链接混合使用)
    • text-indent:50px 设置文本缩进为50px，通常是将段落的首行缩进，允许缩进使用负值

    12. 文字修饰

    ​ CSS关于文字修饰的属性

    • font-family:字体种类 它可以设置多种字体，但在浏览器显示时需要操作系统中本身有该字体，不然会从浏览器下载该字体，多个字体按顺序来优先显示
    • font-style:italic 将文字样式设置为斜体，默认值为normal即正常显示
    • font-size:40px 将文字设置为40像素大小，还要一个单位em 1em=16px
    • font-weight:bold 将文字设置为加粗(bold=700)，默认值为normal即正常显示，还可以使用数字表示100~900之间

    • width:500px; 放在table标签中设置表格宽度为500像素
    • height:500px; 放在table标签中设置表格高度为500像素

    HTML表单的设计与使用

    1. 表单的概念

    ​ 表单在网页中主要负责数据采集功能，主要用于搜集不同类型的用户输入，是人和电脑相互交流的通道

    ​ 一个表单由两个基本组成部分：表单标签和表单域，注意：由开始和结束的表单标签来组成一个表单域，一个页面可以有多个表单区域(通过name属性进行区分，不要重名)，但两个表单之间不支持嵌套

    ​ 表单区域代码示例

    • name属性：定义表单名字，由英文字母定义
    • action属性：指定表单提交后，提交的目的地
    • method属性：有post和get，post打包暗箱传输的数据，而get是通过地址栏传值，不写该属性默认值为get

    ​ 表单标签<form>(双) ：包含了处理表单数据所用CGI程序的URL以及数据提交到服务器的方法

    ​ 下面所提到的标签必须包含在表单域中

    ​ 表单域：包含了文本框、密码框、隐藏域、多行文本框、复选框、单选框、下拉选择框和文件上传框等

    ​ 表单按钮(属于表单域)：包括**提交按钮、复位按钮和一般按钮(即什么功能都没有定义的空按钮，用户可以根据自己情况定义)**；用以将数据传送到服务器上的CGI脚本或者取消输入，还可以用表单按钮来控制其他定义了处理脚本的处理工作

    2. 单行文本框(单)

    ​ 用<input> 标签描述，type属性为text

    ​ 范例：定义一个文本框

    • name属性：指定输入框的名字，表单中每一个标签都有命名，不要有重名，接收时按名字来接收，使用英文字母自行定义，不可以为关键字

    • type属性：用于定义类型，单行文本输入框的type固定为text

    ​ 注意：未加修饰的文本框在不同的浏览器显示的效果是不一样的，如果要所有的浏览器看到同一个效果，则需要使用CSS对其进行修饰

    3. 密码输入框(单)

    ​ 用<input> 标签描述，type属性为password

    ​ 密码框和文本框的区别是在密码框输入时的东西都会被隐藏，但传值的时候还是可以看见

    ​ 范例：定义一个密码框

    • name属性：指定密码框的名字，表单中每一个标签都有命名，不要有重名，接收时按名字来接收，使用英文字母自行定义，不可以为关键字

    • type属性：密码输入框的type固定为password

    ​ 注意：可以在表单中加入表格，使得各标签显示更加有序

    4. 单选框(单)

    ​ 用<input> 标签描述，type属性为radio

    ​ 单选框一次只能选择一个选项，在单选框使用时，必须有两个以上的单选框标签，而且其name属性相同，每一个单选框构成一个选项

    ​ 范例：性别选择的实现

    • name属性：name属性相同表示一组单选框
    • type属性：单选框的type固定为radio
    • value属性：单选框相对于用户来说只是提供选择，用户并没有往里面输入值，所以需要value属性来定义一个值，当发生传值时，将选中的单选框的value值传递出去，单选框传的值和页面显示的文字无关
    • checked属性：没有值，表示该选项默认为选中状态

    ​ 注意：文本框也有value属性，如果写了value属性，页面显示的就为value的默认值

    5. 复选框(单)

    ​ 用<input> 标签描述，type属性为checkbox

    ​ 复选框用于多选

    ​ 范例：城市的选择

    • name属性：name属性相同表示一组复选框
    • type属性：复选框的type固定为checkbox
    • value属性：复选框相对于用户来说也只是提供选择，用户并没有往里面输入值，所以需要value属性来定义一个值，当发生传值时，将选中的复选框的value值传递出去，复选框传的值和页面显示的文字无关
    • checked属性：没有值，表示该选项默认为选中状态

    6. 下拉单选框(双)

    ​ 用<select> 包含<option> 标签来描述，没有type属性

    ​ 下拉单选框一次只能选择一个选项，下拉单选框的本质还是单选，但其可以节省空间，可以选如年月日那种

    ​ 范例：下拉单选框的使用

    • select标签的属性
      • name属性：指定下拉单选框的名字，使用英文字母自行定义
    • option标签的属性
      • value属性：指定该选项的值，传参过去时使用
      • selected属性：没有值，表示默认选中该选项

    7. 多行文本输入框(双)

    ​ 用<textarea> 标签描述，没有type属性

    ​ 多行文本输入框用于输入大量文字信息，用于像电子邮件、发帖的正文等

    ​ 范例：多行文本输入框的实现

    • name属性：指定输入框的名字，用英文表示自行定义
    • rows属性：指定文本框的行数(20表示有20个字符那么长)
    • cols属性：指定文本框的列数(80表示有80个字符那么长)
    • 默认值：在开始标签和结束标签中间写的东西都会作为默认值显示

    ​ 注意：现在一般不会这样使用cols属性和rows属性来确定多行文本输入框的大小

    8. 文件上传(单)

    ​ 用<input> 标签描述，type属性为file

    ​ 用于把文件上传到服务器

    ​ 范例：文件上传的前端部分

    • name属性：指定该文件上传接口的名字，用英文表示自行定义
    • type属性：提交按钮的type属性固定为file

    ​ 注意：此时只是为前台页面的上传，该上传并不具有文件上传的逻辑，要想真正实现上传专门去用程序去实现上传的组件，他只是前台的一个接口

    9. 提交按钮(单)

    ​ 用<input> 标签描述，type属性为submit

    ​ 用于将表单数据提交给form 标签中action属性指定的地址进行处理，即传出去了，但action的那个页面能否接收要看那个页面是怎么写的

    ​ 范例：实现提交按钮

    • name属性：指定按钮名字，用英文字母自行定义
    • type属性：提交按钮的type属性固定为submit
    • value属性：指定按钮上显示的字

    10. 普通按钮(单)

    ​ 用<input> 标签描述，type属性为button

    ​ 该按钮相当于空的按钮，需要自己使用脚本进行定义

    11. 重置按钮(单)

    ​ 用<input> 标签描述，type属性为reset

    ​ 用于清空表单已经填写的数据，把表单恢复初始化

    ​ 范例：定义重置按钮

    • name属性：指定按钮名字，用英文字母自行定义
    • type属性：重置按钮的type属性固定为reset
    • value属性：指定按钮上显示的字

    HTML的标签

    1. 标题标签(双)

    ​ 标题是通过<h1> -<h6> 等标签进行定义的，其中<h1> 定义最大的标题，<h6> 定义最小的标题

    ​ 注意：标题标签只能用于标题，不要为了产生粗体或大号的文本而使用标题，标题是为网页结构和内容编制的索引，搜索引擎会抓取标题当关键字

    2. 基本标签

    • 段落标签(双)

      段落是使用<p> 标签在HTML页面中创建段落

    • 水平线标签(单)

      水平线是使用<hr>标签在HTML页面中创建水平线

    • 换行标签(单)

      换行是使用<br>标签在HTML页面中创建换行符，<br />是写XHTML时的一个严格语法，如果不是写XHTML，则HTML中都应该使用<br>，注意：在代码中敲多少回车都没有用，多个空白符当一个空格

    • 空格标签(符)

      空格是使用&nbsp标签在HTML页面中显示空格

    • 注释

      HTML使用<!-- -->期间的内容为注释，注释在页面不显示，代码中可显示

    • 加粗( 双)

      加粗是使用<b>标签对标签内的文本进行加粗

    3. 图像标签(单)

    ​ 图像使用<img> 标签在HTML页面中创建图像

    ​ <img> 标签常用的属性如下（加粗为必须要使用的属性）：

    • src：设定要引用图像的路径，如果图像路径不正确会在web页面上显示损坏的图像

    • alt：为图像定义一串预备的可替换文本，在浏览器无法载入图像时，浏览器将显示这个替代性文本

    • width：设定图像宽度，属性值为像素

    • height：设定图像高度，属性值为像素

      注意：一般不会采用width或者height这种形式对图像进行强制放缩，因为采用这种形式进行放缩，文件量是不变的，即大图像变小但是文件大小不变，此时会造成网页上有许多占用流量的图像，使得网页反馈速度变慢

    4. 路径

    ​ 路径分为网络路径和硬盘物理路径，我们在网上使用的都是网络路径

    ​ 网络路径分为绝对路径和相对路径

    • 绝对路径：由根目录（/）开始写起的文件名或目录名称
    • 相对路径：相对于目前路径的文件名写法

    ​ 路径的切换

    • 同级别：直接写文件名
    • 上一次级别：用../ 文件名
    • 下一次级别：用目录名/ 文件名

    ​ 注意：在网页中只能使用/来分隔路径，不可以使用\，\是Windows操作系统分隔路径用的

    5. 超级链接（双）

    ​ 超链接使用<a> 标签在HTML页面中创建超链接，开始标签和结束标签之间的文字、图像被作为链接来显示

    ​ <a> 标签里的内容会成为web页面中可单击的文本，默认这是有下划线的，可以用CSS去除

    ​ <a> 标签常用的属性如下（加粗为必须要使用的属性）：

    • href：用于定位需要链接的文档，里面写要跳转网页的路径，或者目标文件，事实上可以指向各种各样的资源

    • target：定义被链接的文档在何处显示

      常用的两个值

      1. _self：在本窗口打开链接，不设置target则此为默认值
      2. _blank：在一个新窗口打开链接

    • title：定义鼠标停留在链接上提示信息

    6. 列表(双)

    ​ 列表分为有序列表和无序列表

    • 无序列表：在HTML中使用<ul> 标签表示无序列表，每一个列表项使用<li> 表示

    • 有序列表：在HTML中使用<ol> 标签表示有序列表，每一个列表项使用<li> 表示

      注意：不管是有序列表还是无序列表<li> 都是双标签，而且这三个标签都为块元素

    • 自定义列表：使用<dl> 、<dt> 、<dd> 来组成自定义列表

      浏览器显示效果

    ​

    ​ 列表可以相互嵌套，但嵌套的标签需要放在<li> 里面

    7. 表格(双)

    ​ 表格一般主要用于展示数据，但在网页设计中，主要用于布局，做容器使用

    ​ 表格的图示

    ​ 在HTML中使用<table> 标签来定义表格；
    ​ 使用<tr> 标签表示表格的一行，<tr> 标签里面不可以直接放元素；
    ​ 使用<td> 来表示表格的一列(必须在行标签中)，即单元格，在<td> 标签中可以包含文本、图片、列表、段落、水平线、表格等任何元素；
    ​ 使用<th> 来表示表头，该标签可加可不加

    ​ 注意：上述标签都为双标签

    ​ <table> 标签的常用属性

    • cellspacing ：用来定义表格的间距(单元格边框与单元格边框之间的距离)，默认值为1px
    • cellpadding ：用来定义表格的边距(内容与单元格边框之间的距离)
    • width ：用来定义表格的宽度
    • height ：用来定义表格的高度
    • border ：用来定义表格的边框粗细

    注意：上述属性都是用像素(px)为单位，一般设计网页设定分辨率<1000px，14寸笔记本分辨率为(1366* 768)px

    • align ：放在<table> 标签中用来定义表格在总页面中的水平对齐方式；放在<td> 标签中用来定义单元格中的内容在单元格中的位置，两个的默认值都为left

    8. 引用(双)

    (1).<q> 标签表示的引用(本质上为行内元素)

    ​ 当需要引用一段文字时可以使用<q> 标签来将该段文字被引用

    ​ 在不同的浏览器中显示情况不同，有些浏览器会为被引用的文字添加双引号(该双引号在网页不会被选中)，而有些浏览器不会，所以在使用时需要自行调试

    (2).<blockquote> 标签表示的引用(本质上为块元素)

    ​ 当需要引用一段文字时可以使用<blockquote> 标签来将该段文字被引用

    ​ 在浏览器中显示一块区域类似<p>但里面的放的文字会有所缩进(有些浏览器可能不会缩进)

    注意：两个标签表示的引用效果相同，只是一个是行内元素，另一个是块元素

    9. 特殊字符

    ​ 在HTML中有些字符用于组成HTML页面，导致要输出其时无法正常显示，此时就需要使用HTML规定的特殊字符表示的方法

    ​ 常用的特殊字符

    1. <字符：&lt
    2. >字符：&gt
    3. &字符：&amp

    10. 原样输出

    ​ 在HTML中使用<pre> 标签来告诉浏览器，标签中的内容按照你输入方式的原样显示文本

    11. 代码显示(双)

    ​ 在HTML中使用<code> 标签来显示计算机程序代码

    HTML简介

    1. 互联网的起源

    ​ 互联网起源于美国，最先是为了服务军方，后面由几个大学不断研发。首先是局域网，几个城市的局域网相互连接变成城域网，几个城域网相互连接变成广域网，而由几个国家连接就为互联网(因特网)。

    ​

    2. W3C组织

    ​ 由万维网发明者蒂姆.伯纳斯.李创立，其创建的目的是为了防止互联网陷入割据，现在很多web标准和技术都是由他制定

    3. HTML的概念

    ​ HTML：即超文本标记语言，是构成网页文档的主要语言
    ​ 超：指的是规定好的关键字，不放浏览器是看不出效果的
    ​ 文本：指的是类似像使用记事本写的都是文本文件
    ​ 标记：指的是由<>组成的标记(标签)
    ​ 语言：HTML是解释执行语言，只是翻译不做逻辑

    ​ HTML页面也被称为静态页面或者HTML文档，后缀为html、htm，由于是之前操作系统只支持三位扩展名，到现在都支持了

    ​ HTML语言通过浏览器进行翻译

    4. HTML的补充说明

    ​ HTML是有版本的，最新的是HTML5，HTML是向下兼容，而且有些旧的浏览器可能不兼容最新版的HTML版本

    ​ 同一个页面在不同浏览器(内核不同)中看到效果可能不同，这也称为浏览器兼容性问题

    5. HTML标签

    ​ HTML标签由尖括号包围的关键字构成

    ​ **HTML标签有成对出现(双标签)**，其中第一个标签是开始标签，第二个标签是结束标签； **也有单个出现(单标签)**，所有的单标签都以/结束

    6. HTML属性

    ​ HTML标签可以拥有属性，属性提供了有关HTML元素的更多信息，属性需要写在开始标签中，一个标签可以有多个属性

    ​ 属性的结构为：名称 = “value”

    7. HTML页面的基本结构

    ​ HTML页面的结构包括头部(Head)、主体(Body)两大部分；头部描述浏览器所需的信息，主体包含所要说明的具体内容

    ​ 标签是可以嵌套的，但不可以随意嵌套，需要明白父子关系

    ​ HTML页面的命名尽量使用英文，不要使用中文

    10. 标签的属性

    ​ 标签属性的作用是提供元素的一些额外信息

    ​ 在HTML中属性的写法为：属性名=”属性值”

    ​ 注意：属性名和属性值都必须为HTML文档规定合法的写法

    11. 元素

    ​ HTML中的元素由三个部分组成：开始标记 + 内容 + 结束标记（单标签为各例）
