  • 建立C(sharp)类库

    2020-11-26 |
    • { 杂记 }

    用VS2019 在C#新建一个算术类库

    1.在VS2019中新建一个类库程序

    2.在编写完成后点击F6在下面的生成中可以看见路径

    3.一般是在bin这个文件夹中，点击进入找到一个dll的文件，此就为生成的类库

    使用生成的类库

    1.新建一个控制台应用程序
    2.在引用中添加引用，在浏览处选择浏览，找到想要引用的类库添加即可

    类库的修改

    由于依赖性类库需要在现有框架下修改，使用文件.csproj结尾的，添加现有项目XXX.csproj即可在文件中修改，即可以看见该类库的源代码，显示后可以在添加引用项目，解决方案中引用这个类库，修改后先移除原先的再重新引用即可生效。

    使用自己类库的方式和系统给的类库的方式一样
