    事件

    1. 事件

    ​ 事件处理是整个JavaScript的核心操作

    ​ 在页面上任何的操作都可能被称为事件源，而对每一个事件可以自定义处理操作，例如页面加载，鼠标移动等都被称为事件源

    ​ 在JavaScript中，事件都是以onXXX的形式命名的

    ​ 最基础的静态事件即在HTML标签中声明需要该事件并定义事件名要加小括号和分号，如XXXX();，而后在JavaScript中定义与事件名相同的函数

    ​ Javascript是基于对象的编程语言，所有的对象都在元素上，即每一个元素都属于一个JavaScript的对象

    2. 页面加载事件与页面卸载事件

    ​ 注意：这两个事件只能够在<body> 元素中进行处理

    (1). 页面加载事件：onload

    ​ 页面加载完成后执行的动作

    ​ 页面加载时，大致可以分为以下几个步骤：

    ​ 1. 开始解析HTML文档结构

    ​ 2. 加载外部样式表及JavaScript脚本

    ​ 3. 解析执行JavaScript脚本

    ​ 4. DOM树渲染完成

    ​ 5. 加载未完成的外部资源（如 图片）

    ​ 6. 页面加载成功

    3. 鼠标事件

    事件类型 说明
    click 单击鼠标左键时发生，如果右键也按下则不会发生。当用户的焦点在按钮上，并按了回车键时，同样会触发事件
    dblclick 双击鼠标左键时发生，如果右键也按下则不会发生
    mousedown 单击任意一个鼠标按钮时发生
    mouseout 鼠标指针位于元素上，且将要移出元素的边界时发生
    mouseover 鼠标指针移入元素时触发
    mouseup 松开任意一个鼠标按钮时发生
    mousemove 鼠标在某个元素上移动时发生

    ​ 范例：使用单击点击事件

    ​ 提升：鼠标经过表格变色

    ​ 注意：bgcolor是HTML，bgColor是JavaScript

    4. 动态设置事件

    ​ 在JavaScript中，事件的绑定除了在元素上使用onXXX 静态的定义之外，还可以动态的设置

    (1).动态设置事件的全过程

    ​ 范例：使用onload在页面加载时对事件进行处理

    ​ 注意：id是整个JavaScript的操作核心所在，必须存在

    ​ 动态设置操作事件的好处在于HTML代码不会和JavaScript代码混合在一起

    5. innerHTML

    ​ JavaScript可使用document.getElementById(id).innerHTML = “新内容”来修改id元素内的HTML内容

    ​ 范例：使用innerHTML

    ​ innerHTML和document.write的区别

    1. document.write：是直接写入到页面的内容流，如果在写之前没有调用document.open, 浏览器会自动调用open。每次写完关闭之后重新调用该函数，会导致网页内容全部被重写
 2. innerHTML：是DOM页面元素的一个属性，代表该元素的html内容。（获取div里面的内容 ，修改内容）你可以精确到某一个具体的元素来进行更改，其不会导致页面全部重绘

    6. 时钟

    ​ 范例：获取时间

    ​ 在JavaScript中，提供setTimeout(函数，时间-毫秒)函数，该函数是指在指定时间后执行一次指定函数，常用作定时器

    ​ 范例：实现页面时钟

    ​ 注意：此时JavaScript代码放在body后面，如果放前面就无法显示

    ​ JavaScript放在head和放在body后的区别

    1. 放在head中：需调用才执行的脚本或事件触发执行的脚本放在HTML的head部分中。当你把脚本放在head部分中时，可以保证脚本在任何调用之前被加载
 2. 放在body后：当页面被加载时立即执行的脚本放在HTML的body部分。放在body部分的脚本通常被用来生成页面的内容。

    JavaScript基础

    1. JavaScript简介

    ​ JavaScript是一种基于对象的编程语言，所有的对象已经由浏览器提供好了；而Java是属于面向对象语言，面向对象语言是强调类的设计

    ​ JavaScript和Java没有本质关系，JavaScript的前身称为LiveScript，后来由于Java在浏览器技术端的发展，后来更名为JavaScript，最新的叫法称为EcmaScript

    ​ 后来发展超出了JavaScript的预期，它后期开始模拟面向对象编程，可以进行服务器端编程(即Node.js)

    2. JavaScript的使用

    ​ JavaScript的基础语法和Java很相似，JavaScript是嵌入在HTML语法之中的编程语言，可以使用”<script> “标记内进行使用，这样就可以实现动态页面的操作效果(前台页面)

    3. JavaScript文件的导入

    ​ JavaScript可以嵌入到HTML文件中使用，但是实际开发过程中，JS的代码是很多的，如果将JS代码写在HTML页面中，就会非常难以维护

    ​ 所以可以单独建立一个*.js的文件，将JavaScript代码写入，然后再将需要的*.js文件进行导入

    ​ 范例：通过src属性导入js文件
