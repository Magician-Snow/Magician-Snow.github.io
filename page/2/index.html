JSP内置对象

1. 内置对象的概念

​ 在Java程序中给了你一个类，要使用他用户需要创建一个对象，而内置对象指的是不需要用户自己创建的对象(即不需要用户自己手工进行对象实例化)。而是由WEB容器在它启动时默认创建好的对象，用户可以直接拿来使用

2. 内置对象

​ 当前主要学习JSP中提供的九个内置对象

序号 对象名称 JavaEEAPI路径 描述
1 pageContext javax.servlet.jsp.PageContext 页面的上下文操作
2 request javax.servlet.http.HttpServletRequest 服务器端接收客户端请求内容
3 response javax.servlet.http.HttpServletResponse 服务器端回应客户端的请求操作
4 session javax.servlet.http.HttpSession 描述每一个用户(登录)的信息
5 application javax.servlet.ServletContext 描述整个当前项目的上下文
6 config javax.servlet.ServletConfig 取得默认的配置信息
7 out javax.servlet.jsp.Writer 进行输出，一般不用
8 page javax.lang.Object 每一个页面都属于Object子类(一般不用)
9 exception java.lang.Throwable 默认情况下，所有的异常都可以由容器处理，处理方式为输出错误信息并结束程序运行

​ 通过JavaEEAPI路径可以查询到每个对象有的方法

​ 注意：在这九个内置对象中，前五个内置对象是精华需要重点了解学习，但要学好JSP开发，以上内置对象需要全部记住

3. 四种属性范围

​ 四种属性范围是整个JavaWeb的核心概念，如果不清楚四种属性范围，就无法更好的理解MVC，也无法进行有效的程序开发，以及后面理解Struts、SpringMVC等概念

​ 属性范围的概念

​ 在任何一个Web项目中都会存在多个页面，如果一个对象需要跨越多个页面后依然可以被使用，这被称为属性的保存范围，在JSP中对属性的操作提供如下三个方法。

  1. 设置属性：public void setAttribute(String name,Object value)

    属性名要String类型且不允许重复(重复就会覆盖)，value是Object类型意味着可以报存所有的Java类型

  2. 取得属性：public Object getAttribute(String name)

  3. 删除属性：public void removeAttribute(String name)

​ 在JSP里面一共有四个对象具备以上的三个方法：pageContext、request、session、application

​ 四种属性范围刚好也是对应了这四个对象

  • display:inline-block 当前元素既可以控制宽高，又可以横向排列

9.浮动与浮动清除

​ 浮动主要是排版使用，可以使得原本2D的网页设计变成3D

​ 浮动可以理解为让某个div元素脱离标准流，漂浮在标准流之上，和标准流不是一个层次

浮动的设置(对于要浮动的盒子设置)

​ CSS中使用float 属性来控制浮动，其默认值为none ，如果将float 属性设置为left 或者right ，元素就会向左侧或者右侧靠紧，同时盒子宽度不再伸展，没设置宽度时会根据盒子里面的内容来确定宽度

​ 如果某块设置为浮动，那么其后面的块会自动补该块未浮动时的位置，但后面块的文字会被挤占

浮动的清除(对于浮动盒子的下面盒子设置)

​ CSS使用clear 属性来清除浮动(消除浮动对下面盒子的影响)，它的取值有left、right、both和none(默认值)，设置为left或者right表示该盒子的左边或右边不允许有浮动的对象，设置为both则表示两边都不允许有浮动对象，因此该盒子将会在浏览器中另起一行显示

10. position属性的使用

​ 一般使用position属性来对盒子模型进行定位，一般不会使用margin来设置位置，因为margin只是指外边距

​ position属性修改元素位置后，对其他元素无影响

​ position属性常用的4个值

  • static

    ​ static是position的默认属性值，忽略top、bottom、left、right

  • relative

    ​ relative表示相对定位，当 position 为 relative 的时候，如果设置了 left 、right、top、bottom 属性，它们会生效，发生偏移时的参照为 position 属性取 static 时盒子的位置

  • absolute

    ​ absolute表示绝对定位，当 position 为 absolute 的时候，元素会被移出正常文档流，并不为元素预留空间，元素相对于最近的非 static 定位祖先元素发生偏移，当元素没有非 static 定位祖先时，会相对 html 发生偏移

  • fixed

    ​ fixed表示固定定位，当 position 为 fixed 的时候，元素会被移出正常文档流，并不为元素预留空间，元素相对于屏幕视口（viewport）的 位置来发生偏移, 元素的位置在屏幕滚动时不会改变

    换行是使用<br>标签在HTML页面中创建换行符，<br />是写XHTML时的一个严格语法，如果不是写XHTML，则HTML中都应该使用<br>，注意：在代码中敲多少回车都没有用，多个空白符当一个空格

  • 空格标签(符)

    空格是使用&nbsp标签在HTML页面中显示空格

  • 注释

    HTML使用<!-- -->期间的内容为注释，注释在页面不显示，代码中可显示

  • 加粗( 双)

    加粗是使用<b>标签对标签内的文本进行加粗

3. 图像标签(单)

​ 图像使用<img> 标签在HTML页面中创建图像

​ <img> 标签常用的属性如下（加粗为必须要使用的属性）：

  • src：设定要引用图像的路径，如果图像路径不正确会在web页面上显示损坏的图像

  • alt：为图像定义一串预备的可替换文本，在浏览器无法载入图像时，浏览器将显示这个替代性文本

  • width：设定图像宽度，属性值为像素

  • height：设定图像高度，属性值为像素

    注意：一般不会采用width或者height这种形式对图像进行强制放缩，因为采用这种形式进行放缩，文件量是不变的，即大图像变小但是文件大小不变，此时会造成网页上有许多占用流量的图像，使得网页反馈速度变慢

4. 路径

​ 路径分为网络路径和硬盘物理路径，我们在网上使用的都是网络路径

​ 网络路径分为绝对路径和相对路径

  • 绝对路径：由根目录（/）开始写起的文件名或目录名称
  • 相对路径：相对于目前路径的文件名写法

​ 路径的切换

  • 同级别：直接写文件名
  • 上一次级别：用../ 文件名
  • 下一次级别：用目录名/ 文件名

​ 注意：在网页中只能使用/来分隔路径，不可以使用\，\是Windows操作系统分隔路径用的

5. 超级链接（双）

​ 超链接使用<a> 标签在HTML页面中创建超链接，开始标签和结束标签之间的文字、图像被作为链接来显示

​ <a> 标签里的内容会成为web页面中可单击的文本，默认这是有下划线的，可以用CSS去除

​ <a> 标签常用的属性如下（加粗为必须要使用的属性）：

  • href：用于定位需要链接的文档，里面写要跳转网页的路径，或者目标文件，事实上可以指向各种各样的资源

  • target：定义被链接的文档在何处显示

    常用的两个值

    1. _self：在本窗口打开链接，不设置target则此为默认值
    2. _blank：在一个新窗口打开链接

  • title：定义鼠标停留在链接上提示信息

6. 列表(双)

​ 列表分为有序列表和无序列表

  • 无序列表：在HTML中使用<ul> 标签表示无序列表，每一个列表项使用<li> 表示

  • 有序列表：在HTML中使用<ol> 标签表示有序列表，每一个列表项使用<li> 表示

    注意：不管是有序列表还是无序列表<li> 都是双标签，而且这三个标签都为块元素

  • 自定义列表：使用<dl> 、<dt> 、<dd> 来组成自定义列表

    浏览器显示效果

7. 表格(双)

​ 表格一般主要用于展示数据，但在网页设计中，主要用于布局，做容器使用

​ 表格的图示

​ 在HTML中使用<table> 标签来定义表格；
​ 使用<tr> 标签表示表格的一行，<tr> 标签里面不可以直接放元素；
​ 使用<td> 来表示表格的一列(必须在行标签中)，即单元格，在<td> 标签中可以包含文本、图片、列表、段落、水平线、表格等任何元素；
​ 使用<th> 来表示表头，该标签可加可不加

​ 注意：上述标签都为双标签

​ <table> 标签的常用属性

  • cellspacing ：用来定义表格的间距(单元格边框与单元格边框之间的距离)，默认值为1px
  • cellpadding ：用来定义表格的边距(内容与单元格边框之间的距离)
  • width ：用来定义表格的宽度
  • height ：用来定义表格的高度
  • border ：用来定义表格的边框粗细

注意：上述属性都是用像素(px)为单位，一般设计网页设定分辨率<1000px，14寸笔记本分辨率为(1366* 768)px

  • align ：放在<table> 标签中用来定义表格在总页面中的水平对齐方式；放在<td> 标签中用来定义单元格中的内容在单元格中的位置，两个的默认值都为left

8. 引用(双)

(1).<q> 标签表示的引用(本质上为行内元素)

​ 当需要引用一段文字时可以使用<q> 标签来将该段文字被引用

​ 在不同的浏览器中显示情况不同，有些浏览器会为被引用的文字添加双引号(该双引号在网页不会被选中)，而有些浏览器不会，所以在使用时需要自行调试

(2).<blockquote> 标签表示的引用(本质上为块元素)

​ 当需要引用一段文字时可以使用<blockquote> 标签来将该段文字被引用

​ 在浏览器中显示一块区域类似<p>但里面的放的文字会有所缩进(有些浏览器可能不会缩进)

注意：两个标签表示的引用效果相同，只是一个是行内元素，另一个是块元素
